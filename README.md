# Aditya Singh — Portfolio

Personal portfolio and resume site. Built with vanilla HTML, CSS, and JavaScript with no frameworks or build tools involved.

Live at https://asingh38-oss.github.io

The site covers my projects, experience, and skills as I finish my CS degree and move into AI/ML engineering full time. It has a light, dark, and auto theme toggle that respects the visitor's system preference by default.

Projects currently featured are Home AI Monitor v2, Moodmetrics, NotebookLM 2.0, and Mock ECU. The whole thing lives in a single index.html file and is hosted on GitHub Pages.
